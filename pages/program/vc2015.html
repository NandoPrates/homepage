Visual Studio Community 2015

O Visual Studio 2015 foi é um conjunto de ferramentas onde você encontra aplicativos para o desenvolvimento de softwares, originalmente desenvolvido pela Microsoft.


Usado para fazer a compilação do emulador através do VC++, utilizando de arquivo de instruções no formato "SLN".

A partir dessa versão o software conta com diversos aplicativos como IDE etc..


Informações adicionais:

Parte do programa é gratuito outras expansões do mesmo são pagas.

Para compra de extensões e mais informações você deve ir a página oficial do produto [visualstudio.com]
