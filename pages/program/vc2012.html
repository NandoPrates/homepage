Visual Studio 2012 Windows Desktop

O Visual Studio 2012 foi é um conjunto de ferramentas onde você encontra aplicativos para o desenvolvimento de softwares, originalmente desenvolvido pela Microsoft.


Usado para fazer a compilação do emulador através do VC++, utilizando de arquivo de instruções no formato "SLN".

Compartilho então essa versão que criei com os pacotes de instalação para que possa instalar offline os produtos.


Como Instalar (Intalador Offline):

Após realizado o download e descompactado abra a pasta e inicie o arquivo wdexpress_full.

Então siga as instruções na tela até o término da instalação.


Como Registrar o produto (Ambas versões):

Com o vc++2012 aberto localize no menu do programa a opção Help depois clique em Register Product, na nova janela clique em Solicitar Chave de ativação.

Será então redirecionado ao website da Microsoft para efetuar cadastro, e uma vez logado siga as instruções do website para receber em seu endereço de e-mail gratuitamente a chave de ativação.

GUARDE A CHAVE SERIAL POIS NÃO PODERÁ SOLICITAR OUTRA EM CASO DE PERDA.


Esse instalador foi preparado por mim, contém todos pacotes para instalar o produto offline para versões windowns 32 e 64 bits.

Se optar pelo instalador online fique atênto que o mesmo pode falhar inumeras veses até de fato conseguir instala-lo.
