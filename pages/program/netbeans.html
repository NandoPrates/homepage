NetBeans IDE

Uma IDE ótima para trabalhar principalmente com php.

Praticalmente esse programa escreve sozinho metade do código de programação.

Facilitando a vida de um programador e fazendo-o ganhar produtividade. Incluzive esse website foi feito usando esse programa.


Originalmente ele foi desenvolvido e projetado para criação de códigos java, mas suas aplicações se superaram, agora ele dar suporte a C, C# entre outros códigos de programação.

E como já mencionei para criar páginas de web seja em html seja em php, pois o mesmo tem instrução para criação através do html 5, css 3, e php 7. Isso mesmo o que á de mais moderno sobre a criação de website terá suporte nessa plataforma de trabalho.

Então não poderia deixar de compartilhar nesse formidável programa.

Porque criar um servidor de ragnarok vai muito além de fazer um download aqui e outro alí, você deve desenvolver códigos de programação e essa é a ferramenta que aconselho quando o assunto for WEB.


Antes de fazer a instalação, você deve assegurar-se que existe instalado em sua máquina o Java JDK.

Realizar downloads de acordo com a arquitetura do seu sistema operacional.
