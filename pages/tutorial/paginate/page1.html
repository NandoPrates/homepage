Instalação do Servidor MySql

O primeiro passo que deve-se dar ao iniciar com um servidor é criar-lo offline.

E ter um programa para gerenciamento dos serviços MySql é indispensável tanto para quem vai criar um servidor de ragnarok quanto para quem já possui algum.

Ou seja esse tipo de programa é principal ferramenta de trabalho, e quando se tem ótimas ferramentas melhor ainda.


Então nesse guia temos o processo de instalação do Xampp em uma máquina windows, solucionando também os problemas de instalção nas versões win 8 e 10.
