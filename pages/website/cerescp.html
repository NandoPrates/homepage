Ceres Control Panel

O CeresCP (Control Painel) é um website para gerenciamento de login e personagens para ragnarok.

De fácil configuração e utilização e muito leve para ser executado pelo servidor.


Suas funções são desde login, cadastro, recuperar senha, como também, como administração desses dados seja pelo administrador, seja pelo próprio usuário.


Para quem não sabe nada sobre o funcionamento ou administração de um web site, o ceres cp é a ferramenta que precisa.

Mas como um expert em PHP devo ser cincero, ele possui muitas falhas de segurança, suas querys podem ser facilmente inserindos o conhecido como sql inject, mas apesar de tudo é uma ótima ferramenta que funciona lindamente.


Recomendo para quem não tem conhecimento de web e vai ter um servidor de pequeno porte.
